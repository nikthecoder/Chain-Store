## Chain Store Database

This repository contains a database schema and SQL queries for managing a chain store's information.

## Database Schema

The database includes the following tables:

- `Assortment`
- `Discount`
- `DiscountCode`
- `Store`
- `BusinessHour`
- `WorkHour`
- `Schedule`
- `Order`
- `OrderProduct`
- `Employee`
- `Inventory`
- `Customer`

## Technologies Used

- Microsoft SQL Server
- T-SQL/MS-SQL
